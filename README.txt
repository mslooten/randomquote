# Random Quote Machine 

Made in 2016 for the freeCodeCamp challenge. Gives a random design quote, along with a random background and text color.

Techniques and resources used:
* HTML5 and CSS3
* JavaSript and jQuery
* Calculates YIQ contrast to determine the quote background (white or dark gray depending on the primary color)
* quotesondesign.com API
* Google Webfonts (Merriweather)

Original: https://codepen.io/mslooten/pen/JKRgYL
